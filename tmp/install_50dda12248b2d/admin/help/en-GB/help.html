Introduction

The SimpleCalendar component for Joomla! CMS aims at providing a simple and lean event list that targets small and medium-sized sports and leisure clubs. This extension only works for Joomla! 1.5 or later.

SimpleCalendar has been translated into many different languages. Please see the Translations section for the list of available translations.
